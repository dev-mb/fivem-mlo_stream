# FiveM | MLO Streaming

Put all MLO Files in the `stream/` Folder.
<br>
Change the Foldername > `mlo_name/` in yout MLO Name & add the Foldername in your `server.cfg`

<br><br>
```
mlo_name/
|-> stream/
|    |-> _manifest.ymf
|    |-> *.ytyp
|    |-> *.ytd
|    |-> *.ydr
|    |-> *.ybn
|    |-> *.ydd
|    \-> *.ymap
\-> fxmanifest.lua
```
